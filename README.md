## Supported Platforms
- macOS 13.3.1
- Apple Silicon Mac

## dependencies
- brew
- wget

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install wget
```

## install
```sh
bash <(wget -qO- https://raw.githubusercontent.com/TakanoTaiga/ros2_m1_native/support_rviz/setup.sh)
```
